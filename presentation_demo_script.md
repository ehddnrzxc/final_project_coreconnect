# 🎬 CoreConnect 시연 영상 시나리오

## 전체 구성 (총 3-4분)

---

## 📋 시나리오 개요

### 테마: "한 평범한 업무일의 CoreConnect 활용기"
**핵심 메시지**: 통합 협업 플랫폼으로 업무 효율성 극대화

---

## 🎯 시나리오 1: 오프닝 & 로그인 (0:00 - 0:30)

### 화면 구성
1. **시작 화면 (3초)**
   - CoreConnect 로고 + 슬로건
   - "통합 협업 플랫폼으로 업무를 더 효율적으로"

2. **로그인 화면 (10초)**
   - 깔끔한 로그인 UI
   - "계정 기억" 체크박스 체크
   - 로그인 버튼 클릭
   - **내레이션**: "아침 출근, CoreConnect로 하루를 시작합니다"

3. **대시보드 도착 (10초)**
   - 홈페이지 대시보드 전면 보여주기
   - 여러 카드들 한눈에 보이도록
   - **내레이션**: "하루의 업무 현황을 한눈에 파악할 수 있습니다"

---

## 🎯 시나리오 2: 근태 관리 & 출근 (0:30 - 1:00)

### 화면 구성
1. **근태 카드 확인 (10초)**
   - 대시보드에서 근태 카드 클릭
   - 출근 시간 표시 확인
   - **내레이션**: "출근은 간단하게 한 번의 클릭으로"

2. **근태 페이지 이동 (10초)**
   - 근태현황 페이지로 이동
   - 주간/월간 통계 확인
   - 조퇴, 지각 등 통계 표시
   - **내레이션**: "근태 현황을 실시간으로 확인하고 관리합니다"

3. **출근 처리 (10초)**
   - 출근하기 버튼 클릭
   - 출근 시간 기록 확인
   - **내레이션**: "출근 처리 완료! 오늘도 화이팅"

---

## 🎯 시나리오 3: 휴가 신청 프로세스 (1:00 - 2:00)

### 화면 구성
1. **휴가 페이지 접근 (10초)**
   - 휴가 페이지로 이동
   - 내 휴가내역 확인
   - **내레이션**: "휴가가 필요할 때, 신청 절차를 시작합니다"

2. **휴가 신청 버튼 클릭 (5초)**
   - "휴가 신청" 버튼 클릭
   - 전자결재 페이지로 자동 이동
   - **내레이션**: "신청 버튼 하나로 전자결재 시스템과 연동됩니다"

3. **휴가 신청서 작성 (20초)**
   - 휴가 신청 폼 작성
   - 기간, 사유 입력
   - **내레이션**: "간단한 정보 입력으로 휴가를 신청합니다"

4. **결재선 지정 & 제출 (15초)**
   - 결재선 선택
   - 제출 버튼 클릭
   - **내레이션**: "결재선을 지정하고 제출하면, 승인 프로세스가 자동으로 시작됩니다"

5. **일정 캘린더 연동 (10초)**
   - 일정 관리 페이지로 이동 (또는 자동 팝업)
   - 승인된 휴가가 캘린더에 추가된 모습
   - **내레이션**: "승인된 휴가는 자동으로 캘린더에 반영되어 팀 전체가 확인할 수 있습니다"

---

## 🎯 시나리오 4: 메일 & 채팅 (2:00 - 2:30)

### 화면 구성
1. **메일 확인 (15초)**
   - 메일 페이지로 이동
   - 받은 메일함 확인
   - 중요한 메일 읽기
   - **내레이션**: "업무 메일을 효율적으로 관리하고 확인합니다"

2. **채팅으로 협업 (15초)**
   - 채팅 페이지로 이동
   - 팀원과 실시간 채팅
   - 파일 전송 기능 (있으면)
   - **내레이션**: "실시간 채팅으로 팀원들과 빠르게 소통합니다"

---

## 🎯 시나리오 5: 알림 & 공지사항 (2:30 - 3:00)

### 화면 구성
1. **상단 알림 확인 (10초)**
   - 상단바 알림 아이콘 클릭
   - 다양한 알림 확인 (결재 승인 요청, 새로운 메일 등)
   - **내레이션**: "중요한 알림을 놓치지 않고 받아봅니다"

2. **공지사항 확인 (15초)**
   - 공지사항 메뉴 클릭
   - 최신 공지사항 확인
   - **내레이션**: "회사의 중요한 공지사항을 빠르게 확인합니다"

---

## 🎯 시나리오 6: 게시판 & 조직도 (3:00 - 3:20)

### 화면 구성
1. **게시판 활용 (10초)**
   - 게시판 페이지로 이동
   - 게시글 작성 또는 조회
   - **내레이션**: "부서별 게시판으로 정보를 공유합니다"

2. **조직도 확인 (10초)**
   - 조직도 페이지로 이동
   - 회사 조직 구조 확인
   - 직원 프로필 확인
   - **내레이션**: "회사 조직 구조와 팀원 정보를 쉽게 확인할 수 있습니다"

---

## 🎯 시나리오 7: 관리자 기능 (3:20 - 4:00)

### 화면 구성
1. **관리자 페이지 접근 (5초)**
   - 관리자 권한으로 로그인 (또는 계정 전환)
   - 관리자 페이지로 이동

2. **대시보드 통계 (10초)**
   - 사용자 수, 부서 수 등 통계 확인
   - **내레이션**: "관리자는 전체 시스템 현황을 한눈에 파악합니다"

3. **사용자 관리 (10초)**
   - 사용자 목록 확인
   - 새 사용자 생성 또는 사용자 정보 수정
   - **내레이션**: "사용자 관리 기능으로 인사 관리가 쉬워집니다"

4. **비밀번호 초기화 (5초)**
   - 비밀번호 초기화 요청 확인 및 처리
   - **내레이션**: "비밀번호 초기화 요청을 효율적으로 처리합니다"

5. **로그인 이력 (10초)**
   - 로그인 이력 확인
   - **내레이션**: "시스템 접근 이력을 추적하여 보안을 강화합니다"

---

## 🎯 시나리오 8: 마무리 (4:00 - 4:15)

### 화면 구성
1. **종합 화면 (10초)**
   - 여러 페이지를 빠르게 전환하며 보여주기
   - 또는 대시보드로 돌아와서 전체 기능 요약

2. **엔딩 (5초)**
   - CoreConnect 로고
   - "업무를 더 효율적으로, CoreConnect와 함께"
   - 팀 정보 (선택사항)

---

## 💡 촬영 팁

### 화면 녹화 시 주의사항
1. **마우스 움직임**
   - 부드럽고 자연스럽게
   - 중요한 버튼에서는 잠시 멈춤
   - 클릭 시 약간의 딜레이

2. **화면 전환**
   - 너무 빠르지 않게
   - 각 화면에서 2-3초 정도 머무름
   - 페이지 로딩은 건너뛰거나 가속

3. **줌 인/아웃**
   - 중요한 부분은 줌 인
   - 전체 구조 보여줄 때는 줌 아웃

4. **더미 데이터**
   - 현실적인 더미 데이터 준비
   - 실제 업무 시나리오처럼 구성

### 내레이션 팁
1. **톤**
   - 친근하고 전문적인 톤
   - 너무 빠르지 않게, 명확하게

2. **강조점**
   - 핵심 기능 강조
   - 사용자 이점 언급

3. **자연스러움**
   - 딱딱하지 않게
   - 스토리텔링 요소 포함

- [ ] 마이크 음질 확인

## 🎯 핵심 포인트

1. **통합성 강조**: 모든 기능이 하나의 플랫폼에서 작동
2. **사용자 편의성**: 간편한 클릭 몇 번으로 업무 처리
3. **자동화**: 휴가 신청 → 전자결재 → 캘린더 연동
4. **실시간성**: 근태, 알림, 채팅의 실시간 업데이트
5. **관리 효율성**: 관리자 대시보드의 통합 관리
